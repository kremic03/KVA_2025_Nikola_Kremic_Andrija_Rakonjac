About CinemaTickets

Welcome to our digital cinema ticketing system prototype!

Project Description

This application is a web app prototype for browsing and booking movie tickets. It was developed for the Client Web Applications course at Singidunum University.

Features

  • Browse and search movies by various criteria
  • View detailed movie information
  • Make ticket reservations
  • Manage user profile
  • Track reservation history
  • Rate watched movies

Technology Stack

  • Angular 19
  • Angular Material
  • TypeScript
  • HTML/CSS
  • Axios for HTTP requests
